# OdontoCloud - React Starter (Vite)

Estructura inicial del proyecto OdontoCloud creada por ChatGPT.
- Proyecto listo para abrir con `npm install` y `npm run dev` (usa Vite).
- Contiene módulos base vacíos para que los vayas llenando: agenda, pacientes, facturacion, odontograma, inventario, reportes, administracion.
- Incluye placeholders para Firebase y ejemplos de carga dinámica de módulos.

**Nota:** Este es un esqueleto. Contenido visual y estilos mínimos a propósito para que conserves tu diseño original.
